Airlines

  • Project Role: Developer
  • Project date: 2013 - 2015
  • Project Specific Skills: AWS Redshift, Analysis tool for reporting, Amazon S3, Micro strategy, Amazon EMR, Data Pipeline, Spark

Description

The Airline data is downloaded from the given source. Ten years of historical data is analysed to extract necessary information. The data is extracted and loaded into Hive tables after which Java programs are used for cleaning and parsing the data. Hive and Impala are used to query the data and display the results on web UI using MicroStrategy.
